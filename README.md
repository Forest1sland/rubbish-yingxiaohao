# 打击营销号

本项目意在通过插件和名单的方式，打击营销号，防止小白避坑，维护更好的知识生态。

## 怎么识别营销号？

内容纯搬运、洗稿、抄袭、无任何原创，声音一般为语音合成，置顶评论一般为让你加群，加了你就上当了
